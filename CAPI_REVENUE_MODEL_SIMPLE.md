## 3 Tipos de Clientes

Capi tendrá 3 tipos de clientes. Cada uno genera ingresos de forma diferente:

### 1️⃣ ONGs Pequeñas (80% de usuarios)

**¿Quiénes son?**
- Pequeñas organizaciones sin fines de lucro
- 2-50 personas
- Coordinan eventos puntuales (limpiezas, talleres, etc.)
- Ubicadas en Latinoamérica

**Cómo usan Capi:**
```
Coordinadora María (ONG pequeña):
│
├─ Crea evento: "Limpieza de parque - sábado 10am"
├─ Invita voluntarios: Por Capi o WhatsApp
├─ Recibe confirmaciones automáticas
├─ Hace seguimiento de asistentes
├─ Realiza el evento
└─ Paga: $0 (GRATIS)
```

**¿Capi gana dinero aquí?**
- ❌ NO directamente
- ✅ PERO gana users, datos, credibilidad

**¿Cuántos hay?**
- Año 1: 2,000 ONGs
- Año 3: 30,000 ONGs
- Año 5: **75,000 ONGs** 👑

---

### 2️⃣ ONGs Medianas (15-20% de usuarios pagan)

**¿Quiénes son?**
- ONGs más formales
- 50-500 personas
- Presupuesto pequeño pero existen
- Quieren reportes y medición

**Cómo usan Capi:**
```
Coordinadora Paula (ONG mediana):
│
├─ Usa TODO lo de ONG pequeña (gratis)
├─ Pero además paga $29/mes por:
│  ├─ Reportes profesionales (gráficos, datos)
│  ├─ Certificados digitales para voluntarios
│  ├─ Integración con Google Calendar
│  ├─ Recordatorios automáticos por email
│  ├─ Análisis de impacto social
│  └─ Support por email prioritario
│
├─ Paga: $29/mes = $348/año
└─ Beneficio: Todo lo que necesita una ONG mediana
```

**¿Capi gana dinero aquí?**
- ✅ SÍ, pero es ingresos pequeños
- 10-20% de usuarios free suben a pago

**Proyección:**
```
AÑO 2:      200 ONGs × $290/año = $58,000
AÑO 3:    1,000 ONGs × $290/año = $290,000
AÑO 5:    7,500 ONGs × $290/año = $2,175,000
```

**¿Quiénes son?**
- Empresas con 100+ empleados
- Con presupuesto CSR/voluntariado
- Ubicadas en Latam (México, Colombia, Perú, etc.)

**Cómo trabaja Capi con ellos:**

```

│  │  ├─ Estrategia: Alinear con Objetivos Sostenibles (ODS)
│  │  ├─ Selección: Qué ONGs partner usar
│  │  ├─ Capacitación: Entrenar coordinador de empresa
│  │  └─ Setup: Configurar plataforma Capi
│  │
│  └─ MES 4-12: GESTIÓN MENSUAL (Recurrente)
```

**¿Capi gana dinero aquí?**
